SYNC · Diseño y Desarrollo Web

SYNC es un proyecto para un festival ficticio dedicado al cine independiente y experimental. La identidad visual combina dinamismo y contemporaneidad, diseñada para un público joven, creativo y curioso. Durante el proceso de investigación y diseño se tuvieron en cuenta antecedentes como el Edinburgh Film Festival o Tribeca Film Festival.

El objetivo era crear una página web estática atractiva y funcional usando únicamente HTML, CSS Y JavaScript. Se llevó a cabo mediante el uso de modales y hover effects.
